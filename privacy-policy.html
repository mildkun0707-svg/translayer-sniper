Privacy Policy – TransLayer Sniper

Last updated: April 2026

1. Overview

TransLayer Sniper ("the Extension") is a Chrome browser extension that helps users translate official documents using AI. This policy explains what data is collected, how it is used, and your rights.

2. Data We Collect

  • API Keys – Your Google Gemini API key and (optionally) Anthropic Claude API key, entered by you. Stored locally in your browser using chrome.storage.local. Never transmitted to our servers.
  • Document content for translation – Screenshots (image data) or selected text from the webpage you are viewing are sent directly from your browser to Google Gemini API or Anthropic Claude API for translation processing. We do not receive or store this content.
  • Translation history (Ledger) – Saved locally in your browser only. Never uploaded to any server unless you explicitly configure Supabase integration.
  • Translation memory – Custom terminology pairs you create, stored locally in chrome.storage.local.
  • Supabase integration (optional) – If you choose to enable Supabase, translation terms may be synced to your own Supabase project using credentials you provide. The Extension developer has no access to your Supabase data.

3. Data We Do NOT Collect

  • We do not collect personal information (name, email, location).
  • We do not track browsing history or page visits.
  • We do not share any data with third parties beyond the AI APIs (Google Gemini, Anthropic Claude) which you have configured.
  • We do not run any analytics or tracking scripts.

4. Third-Party Services

The Extension communicates with the following external services, solely to perform translation:

  • Google Gemini API – Google Privacy Policy
  • Anthropic Claude API (optional, for scoring) – Anthropic Privacy Policy
  • Supabase (optional, user-configured) – Supabase Privacy Policy

Document content sent to these APIs is subject to each provider's own privacy policy. We recommend reviewing them before use.

5. Why <all_urls> Permission Is Required

TransLayer Sniper's core function is to allow users to select and translate any text or image on any webpage — including PDF viewers, government portals, and visa application sites. The <all_urls> host permission is required solely to inject the translation UI overlay and capture selected content on any site the user chooses to translate. The Extension does not access, read, or transmit page content unless the user explicitly activates Sniper Mode.

6. Data Security

All API keys and translation data are stored exclusively in chrome.storage.local on your device. The Extension has no backend server and cannot access your data remotely.

7. Children's Privacy

This Extension is not directed at children under 13. We do not knowingly collect data from children.

8. Changes to This Policy

We may update this policy. Continued use of the Extension after changes constitutes acceptance of the updated policy.

9. Contact

For questions about this privacy policy, please contact us via the Chrome Web Store support page for TransLayer Sniper.

プライバシーポリシー – TransLayer Sniper

最終更新：2026年4月

1. 概要

TransLayer Sniper（以下「本拡張機能」）は、AIを利用して公文書の翻訳を支援するChrome拡張機能です。本ポリシーでは、収集するデータ、その利用方法、およびユーザーの権利について説明します。

2. 収集するデータ

  • APIキー – ユーザーが入力したGoogle Gemini APIキー、およびオプションのAnthropic Claude APIキーは、ブラウザの chrome.storage.local にローカル保存されます。開発者のサーバーへは一切送信されません。
  • 翻訳対象のドキュメント内容 – スナイパーモードで選択した画像データやテキストは、翻訳処理のためにGoogle Gemini APIまたはAnthropic Claude APIへ直接送信されます。開発者はこのデータを受信・保存しません。
  • 翻訳履歴（台帳） – ブラウザのローカルストレージにのみ保存されます。Supabase連携を有効にしない限り、外部へ送信されません。
  • 翻訳メモリ – ユーザーが作成した用語ペアは chrome.storage.local にローカル保存されます。
  • Supabase連携（任意） – ユーザーが設定した場合のみ、翻訳用語がユーザー自身のSupabaseプロジェクトに同期されます。開発者はSupabaseのデータにアクセスできません。

3. 収集しないデータ

  • 氏名・メールアドレス・位置情報等の個人情報は収集しません。
  • 閲覧履歴やページアクセス情報は追跡しません。
  • 設定済みのAI API（Google Gemini、Anthropic Claude）以外の第三者へデータを共有しません。
  • アナリティクスやトラッキングスクリプトは使用していません。

4. 利用する外部サービス

本拡張機能は翻訳処理のためにのみ、以下の外部サービスと通信します：

  • Google Gemini API – Googleプライバシーポリシー
  • Anthropic Claude API（任意・採点機能用）– Anthropicプライバシーポリシー
  • Supabase（任意・ユーザー設定）– Supabaseプライバシーポリシー

5. <all_urls> 権限が必要な理由

本拡張機能の中核機能は、政府ポータル・ビザ申請サイト・PDFビューアを含むあらゆるウェブページ上で、ユーザーがテキストや画像を選択して翻訳できるようにすることです。<all_urls> のホスト権限は、翻訳UIオーバーレイの表示と選択コンテンツの取得のためにのみ使用されます。ユーザーがスナイパーモードを明示的に起動しない限り、ページの内容へのアクセス・読み取り・送信は一切行いません。

6. データのセキュリティ

すべてのAPIキーおよび翻訳データは、ユーザーのデバイス上の chrome.storage.local にのみ保存されます。本拡張機能にはバックエンドサーバーがなく、開発者がユーザーのデータにリモートアクセスすることはできません。

7. お問い合わせ

本プライバシーポリシーに関するご質問は、Chrome Web StoreのTransLayer Sniperサポートページよりご連絡ください。
